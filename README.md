## 📂 Struktur Proyek

Monorepo ini menyatukan 3 proyek kreatif ke dalam sub-direktori berikut:

### 1. [Rainbow-Clock](file:///C:/Users/acer/AppData/Local/Temp/kreativitasku_combine/kreativitasku/Rainbow-Clock)
*   **Deskripsi**: Aplikasi jam digital berbasis web yang menampilkan efek transisi warna latar belakang pelangi (rainbow gradient transition) secara dinamis dan real-time.
*   **Teknologi**: HTML5, Vanilla CSS3 (Keyframes & Gradients), dan JavaScript (Date Object & DOM manipulation).
*   **Cara Menjalankan**: Cukup buka file `Rainbow-Clock/index.html` langsung di peramban web (browser) Anda.

### 2. [Kalkulator](file:///C:/Users/acer/AppData/Local/Temp/kreativitasku_combine/kreativitasku/Kalkulator)
*   **Deskripsi**: Program kalkulator matematika interaktif berbasis antarmuka baris perintah (CLI - Command Line Interface). Mendukung operasi aritmatika dasar hingga tingkat lanjut.
*   **Teknologi**: Bahasa Pemrograman **C** (`kalkulator.c`).
*   **Cara Menjalankan**:
    1.  Buka terminal/command prompt.
    2.  Lakukan kompilasi menggunakan GCC:
        ```bash
        gcc Kalkulator/kalkulator.c -o kalkulator
        ```
    3.  Jalankan program:
        ```bash
        # Windows
        ./kalkulator.exe
        
        # Linux / macOS
        ./kalkulator
        ```

### 3. [Perkakas_saya](file:///C:/Users/acer/AppData/Local/Temp/kreativitasku_combine/kreativitasku/Perkakas_saya)
*   **Deskripsi**: Direktori penyimpanan personal yang memuat berkas-berkas multimedia kustom (gambar dan audio) serta catatan personal.
*   **Isi Direktori**: Gambar perayaan (*Pentol Anniversary*), file audio (*videoplayback*), dan file dokumentasi personal.
